# Git Training - 電卓

Git ワークフローを練習するための、意図的なバグがある電卓アプリです。

## 使い方

1. ブラウザで `index.html` を開いてください
2. 電卓を使ってみてください - バグがあることに気づくはずです！

## 修正すべきバグ

この電卓には **3つのバグ** があります。それぞれ1行の簡単な修正で直せます。

### バグ 1: 引き算の関数が壊れている
- **ファイル:** `calculator.js`
- **行:** 41
- **問題:** 引き算の関数が足し算をしている
- **修正:** `+` を `-` に変更

### バグ 2: 掛け算の関数が壊れている
- **ファイル:** `calculator.js`
- **行:** 47
- **問題:** 掛け算の関数が割り算をしている
- **修正:** `/` を `*` に変更

### バグ 3: 数字ボタンの表示が間違っている
- **ファイル:** `index.html`
- **行:** 20
- **問題:** 9のボタンが `6` と表示されている
- **修正:** `6` を `9` に変更

## Git ワークフローの練習

1. リポジトリをクローン: `git clone <リポジトリのURL>`
2. 新しいブランチを作成: `git checkout -b fix/あなたの名前-bug-fix`
3. 上記のバグを1つ修正
4. 変更をステージング: `git add .`
5. 変更をコミット: `git commit -m "Fix: 修正内容の説明"`
6. ブランチをプッシュ: `git push origin fix/あなたの名前-bug-fix`
7. GitHub でプルリクエストを作成
